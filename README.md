# Travel Expenses Tracker

A **Travel Expenses Tracker Website** that helps users plan trips, manage budgets, and track expenses efficiently.

---

## Features

- **AI-Powered Itinerary**: Generate a personalized itinerary for your chosen destination.  
- **Expense Logging**: Add and categorize expenses such as food, transport, and accommodation.  
- **Budget Tracking & Alerts**: Set budgets and receive alerts when limits are exceeded.  
- **Reports & Analytics**: View detailed reports on spending per category.

---

## Tools & Technologies

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Node.js, Express.js  
- **Authentication**: JWT (JSON Web Token)  
- **Database**: MySQL  
- **Charts & Visualization**: Chart.js
